# 🏗️ SystemDesign AI

**SystemDesign AI** is an AI-powered system design mentor that helps engineers explore how large-scale distributed systems are architected.
It generates architecture explanations, visual system diagrams, infrastructure insights, scalability estimates, and realistic interview feedback — all in one interactive interface.

The project is designed to make complex system design concepts easier to understand while also simulating the experience of a real system design interview.

---

# 🚀 Overview

Modern software systems such as Netflix, Uber, Spotify, and WhatsApp operate at massive scale. Designing such systems requires careful planning around scalability, reliability, distributed services, and infrastructure components.

SystemDesign AI assists with this learning process by automatically generating system design breakdowns for real-world platforms. It combines large language models with an interactive interface to help users visualize architectures and practice system design thinking.

The application allows users to:

* Generate **clear system architecture explanations**
* Visualize **architecture diagrams dynamically**
* Understand how **infrastructure components** like Redis, Kafka, and CDN fit into the design
* Estimate **system scalability requirements**
* Practice **system design interview questions**
* Receive **structured feedback on their answers**

---

# ✨ Features

### 🧠 AI System Architecture Explanations

Generates structured explanations of how large-scale systems work, including:

* Core services
* Request flow
* Databases and storage
* Scaling strategies
* Reliability considerations

---

### 🏗️ LLM-Generated Architecture Diagrams

Creates architecture diagrams dynamically using Graphviz based on system flow generated by the LLM.

Example flow:

User → CDN → Load Balancer → API Gateway → Microservices → Cache → Database

These diagrams help visualize the interactions between distributed components.

---

### ⚙️ Infrastructure Breakdown

Explains the role of important infrastructure technologies used in modern systems:

* **Redis** for caching and latency reduction
* **Kafka** for event streaming and asynchronous processing
* **CDN** for global content delivery and reduced latency

---

### 📊 Scalability Calculator

Provides a simplified estimation of system capacity requirements based on expected user traffic.

The calculator estimates:

* Requests per second (RPS)
* Backend server requirements
* Peak traffic scaling assumptions

This helps users understand how large systems scale with growing user bases.

---

### 🎯 System Design Interview Mode

SystemDesign AI also acts as a **mock system design interviewer**.

The application:

1. Generates a realistic follow-up system design question
2. Allows the user to submit their answer
3. Evaluates the response using the LLM
4. Provides structured feedback including:

   * Score (0–10)
   * Strengths in the answer
   * Missing considerations
   * Suggestions for improvement

This makes it useful for **interview preparation**.

# 🛠️ Technology Stack

SystemDesign AI is built using modern AI and visualization tools:

* **Streamlit** — interactive web interface
* **LangGraph** — orchestration of multi-step AI workflows
* **LangChain** — LLM integration
* **Groq LLM (LLaMA-3.3-70B)** — fast inference for architecture reasoning
* **Graphviz** — dynamic architecture diagram generation

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/SystemDesign-AI.git
cd SystemDesign-AI
```

Install dependencies:

```
pip install -r requirements.txt
```

Set your Groq API key:

```
export GROQ_API_KEY="your_api_key_here"
```

Run the application:

```
streamlit run app.py
```

The Streamlit interface will open in your browser.
